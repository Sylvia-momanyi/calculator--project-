# Modern Calculator App

This is a simple and modern calculator built using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, sqrt, log, ln)
- Memory functions (MC, MR, M+, M-)
- Keyboard support
- Calculation history
- Modern UI design

## Technologies Used
- HTML
- CSS
- JavaScript
